## 二、方案归类

按主导传感器分为三类：

- **纯视觉**（无 RTK / 无 LiDAR）：YUKA Mini Vision
- **视觉 + RTK**：Navimow i108 / i206 AWD / X420
- **LiDAR 主导**（多数辅以视觉）：Navimow i210 / Ecovacs O1200 / Ecovacs A3000 / MOVA LiDAX Ultra 1000

## 四、与 SLAM/VIO/3DGS 算法相关的关键观察

- **唯一显式标注 VIO 的是 Navimow X420**：其余多用 "EFLS / HoloScope / UltraView" 等品牌名包装，未明确说 VIO；但 LiDAR + IMU + 视觉的方案大概率也跑了某种 LIO/VIO 融合，需通过实测推断。
- **双 LiDAR 是 Ecovacs 的差异化**：360° 旋转 LiDAR 用于建图、3D-ToF 用于近距离避障，分工清晰；这种组合是否真的优于单 360° LiDAR 值得专项对比。
- **MOVA LiDAX Ultra 1000 的 70 m LiDAR 检测距离偏长**：意味着 LiDAR 选型可能更接近室外级（不是消费级 ToF），值得拆解关注雷达型号与点频。
- **YUKA Mini Vision 是图中唯一纯视觉、无 RTK 的产品**：和我们正在做的视觉通道建图最有可比性，建议优先拆其视觉 SLAM 表现。
- **Navimow i210 LiDAR vs MOVA LiDAX Ultra 1000**：均为 1000 m² + LiDAR 主导，可作为同档次的横向对标对象。

## 五、建议优先关注顺序（按目前算法工作）

1. **MOVA LiDAX Ultra 1000**：LiDAR + 视觉、无 RTK，与我们 LiDAR 路线最直接对标。
2. **Segway Navimow X420**：VIO + 视觉 + ToF，与我们视觉通道建图思路最接近。
3. **Mammotion YUKA Mini Vision**：纯视觉、无 RTK，是视觉路线极端方案。
4. **Ecovacs A3000 / O1200**：双 LiDAR 方案，可拆双雷达分工与协同策略。
5. **Segway Navimow i210**：单 3D LiDAR 标杆，结构相对简洁。
