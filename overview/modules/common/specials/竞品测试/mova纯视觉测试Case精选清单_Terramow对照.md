## 三、性能专项（黑盒增量，针对算力疑虑）

原 Terramow 文档以功能与精度为主，下列项**不依赖 USB**，用于从外部推断「是否吃紧、是否随时间变差」。

| 编号 | Case | 测试方法 | 建议记录指标 |
|------|------|----------|----------------|
| **P1** | **障碍物响应延迟** | 直行时前方约 1m 突入低矮障碍；**≥120fps，优先 240fps** 录像 | 障碍稳定出现 → 明显减速/停刀/转向的帧差 → **ms**；≥5 次取均值 |
| **P2** | **动态横穿避障** | 人约 1.5 m/s 横穿前方 | 是否避让成功、是否剐蹭、动作是否犹豫反复 |
| **P3** | **冷启到开始作业** | 断电重启 → 下发作业 → 刀盘实质工作 | **秒**；重复 ≥5 次均值 |
| **P4** | **重定位恢复耗时** | 与 **四-1** 合并：放下机器 → 恢复正常割草无持续转圈 | **秒**；≥3 次 |
| **P5** | **长时作业性能** | 连续作业 ≥3 h（或至低电量）；**T0 / 1h / 3h** 各做一次 **P1**；可选红外测壳温 | 延迟是否变差、停走/App 滞后、异常报警、温度或主观热感 |
| **P6** | **App 位姿刷新体感** | 作业中观察图标更新是否卡顿、跳变 | 主观 + 可选 30s 内计数估算 Hz |

**建议优先级**：**P1、P4、P5** 与算力/热衰减关联最直接，时间紧时优先保留。
