## 2. 测试项清单（细粒度）

### V-T-001 建图后桩上视觉深度学习（沿边一圈后桩上离线学习）

- **场景 / 专题**：建图后处理；离线视觉建图优化
- **涉及竞品**：
  - 九号 X3：22% → 100% 用 75min（1800㎡）；学习后宣称"更贴边"，但实测建图未贴边、最终也未见明显贴边提升 [26.1 §Navimow X3 - 建图后视觉深度学习]、[海外分享]
  - 九号 i108：UI 显示需 125min，实际地图小则用不到 2 小时 [26.1 §Navimow i108 - 建图后桩上视觉深度学习]
- **关键指标**：学习时长、覆盖面积、贴边精度提升量（**未量化**）
- **触发逻辑**：仅出基站时学习，再出桩则不学习（[海外分享]）
- **未解**：复割回桩后是否会再次学习（[海外分享] 明确未确认）

### V-T-002 EFLS 视觉建图优化（在线触发）

- **场景 / 专题**：建图后处理；融合一致性补救
- **涉及竞品**：
  - 九号 i206 RTK 款：用户可选开启；信号好时建图后无优化；信号差时触发优化但优化时间仅几分钟、优化后地图基本未变（尤其搬运情况）[方案 §4.1]
- **关键指标**：触发条件（RTK 信号差）、优化耗时、优化前后地图差异（**结论：基本无差异**）
- **限制**：仅 RTK 款有 EFLS

### V-T-003 AI 自主建图 — 起点识别 & 沿边探索

- **场景 / 专题**：AI 自主建图；视觉边界识别
- **涉及竞品**：
  - 九号 (i206)：识别"机器在已建草坪内"以点亮"标记起点"按钮；点亮距离约 1-2m，实测不稳定，有时遥控至草坪内 3-4m 才识别；建图方式 = 沿边探索 + 非目心方式弓字内部探索 [26.3 §2.7 - 设置起点方式 / 找边]
  - 松灵 Luba：AI 建图先做 360° 重定位；边界模糊即转手动；速度慢、不可靠 [26.1 §松灵 - Luba AI 建图展示（效果不好）]、[海外分享]
- **关键指标**：起点识别成功距离、边界跟随成功率（**未量化**）

### V-T-004 AI 自主建图 — 异常处理

- **场景 / 专题**：AI 建图鲁棒性
- **涉及竞品**：
  - 九号：悬崖翻车后放平 → 主动重定位（向前 1m）；若重定位识别在已建草坪外则无法续建（无论手动还是自动）；建图中切割草/回充仍可手动续建，机器先回到上次自动建图结束位置（实际会回退一部分判断"建得不好"的路线）[26.3 §2.7.异常处理]
- **关键指标**：翻车主动重定位距离 = 1m；续建成功条件
- **图片证据**：[26.3 §2.7] image-35/36/37（"转手动失败，要放回"）

### V-T-005 视觉重定位动作（磨草）

- **场景 / 专题**：视觉触发的原地重定位
- **涉及竞品**：
  - 松灵 Luba：原地掉头式重定位，"非常磨草"，弓字也可选该动作 [26.1 §松灵 - 重定位动作]、[海外分享]
- **关键指标**：草地损伤程度（定性："磨草严重"）；动作时长（**未量化**）

### V-T-006 视觉前端的纯定位精度（**本批材料几乎缺位**）

- **场景 / 专题**：VIO / VSLAM 漂移率
- **涉及竞品**：全部仅有现象级描述，**无任何定量**：
  - 九号 X3 RTK 盖锅子时疑似 VSLAM 主导，地图偏 1-2m 但无量化（[26.1 §Navimow X3 - RTK 无信号逻辑探索]）
  - Luba 360° 面阵激光开阔场景偏差 ~2 个机身（[海外分享]，但属激光，非视觉）
- **状态**：**未量化**，列入 next-round 缺口。

### V-T-007 单目辅助避障/测距对运动估计的间接影响

- **场景 / 专题**：单目能力边界（仅作为定位间接背景）
- **涉及竞品**：
  - 九号 i108：人在屁股后拉机器、机器持续往前走 → 排除扭矩检测主导；推测为"单目避障 + 打滑检测 + 过流检测" [26.1 §Navimow i108 - 扭矩/视觉避障]、[海外分享]
- **关键指标**：单目对未触发 bumper 障碍的提前响应距离（**未量化**）
- **本批仅作为现象记录，与定位精度无直接耦合**。
