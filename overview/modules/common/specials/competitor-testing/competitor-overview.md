## 2. 各竞品定位特色 / 已知短板（基于本批材料）

### 九号 X3

- **特色**：RTK + 单目 + 建图后视觉深度学习；信号差时疑似用机身单点 GPS（全频 1.5m 级）做跳变兜底校准；建图无大小限制、地图可重叠保存可手动合并。
- **短板**：边界/禁区不严格（红色实际路径常出界，画了禁区也掉坑）；视觉学习宣称"更贴边"但实测未见明显提升；安全边界判定较松（拉出 3m 内可自回界内）。
- 来源：[26.1 §Navimow X3]、[海外分享]

### 九号（通用 — 退桩 / 通道 / 回桩）

- **特色 / 阈值**：
  - **退桩建图要求 RTK = 4（信号绝对好）才支持** [产品反馈 2026-04-18 §1.1]
  - **草坪间通道**：建通道时对**通道起点定位要求高**，需重定位成功才能建；但即使信号不佳（铁锅盖住基站）**也能重定位成功** [产品反馈 2026-04-18 §1.2]
  - **回桩**：RTK 信号特别不好时大部分情况能正常回充 [产品反馈 2026-04-18 §1.3]
- **短板**：
  - **回桩通道只支持一条**充电桩↔草坪通道（拓扑约束）[产品反馈 2026-04-18 §1.1]
  - **RTK 差 + 搬运**耦合下，偶尔会方向混乱出界 [产品反馈 2026-04-18 §1.3]

### 九号 i 系列两驱（i105 / i108 / i1108）

- **特色**：建图后桩上视觉深度学习（显示 125min）；扭矩 + 单目 + 打滑组合避障（口述：拉机器扭矩检测不出来，但单目能识别杆 → 推测**单目辅助避障 + 过流 + 打滑**而非纯扭矩）。
- **短板**：扭矩检测策略不可靠；定位测试细节本批材料几乎为空。
- 来源：[26.1 §Navimow i108]、[海外分享]

### 九号 i206 (1+1 RTK 三轮)

- **特色**：RTK 良好场景下搬运 / 转圈 / 朝向变把，。化都能实时更新轨迹与朝向，地图可用；EFLS 视觉优化用户可选开启（仅 RTK 款）。
- **短板**：RTK 阴影区下搬运后定位不更新，视觉优化后地图整体偏 1m 不可用；EFLS 信号差时虽触发但优化前后地图基本无变化。
- 来源：[26.3 §2.4 重定位、续建]

### 九号 i215

- **特色**：面阵激光 + RGB-D；本批仅表头登记，无定位实测。
- 来源：[26.3 §竞品信息表]

### 九号 i2 (激光款)

- **特色**：出界行为与 RTK 款一致（慢推 3m 报错；抬后轮拉超 3m 后会做重定位再回充）。
- 来源：[26.3 §15 出界 - 九号激光款 i2]

### 松灵 Luba mini AWD LiDAR

- **特色**：360 激光 + 双目 + nRTK；建图过程中搬运 / 拖拽轨迹和朝向实时更新；3D 实景图。
- **退桩 / 信号差场景策略**（产品观察）：
  - 退桩后做**三点定位**，收集丰富的桩前信息 [产品反馈 2026-04-18 §2.1]
  - 在车库等信号差场景下，**退桩后会一直退**（摄像头朝前、屁股朝后，对它而言是正向走），直到成功 [产品反馈 2026-04-18 §2.2]
- **短板**：
  - 120° 面阵激光在开阔场景定位偏差明显，建图回到原点偏 ~2 个机身、闭合不了，需顶围栏打滑校准（[海外分享]）；
  - 建通道时定位偏差导致建通道失败、需遥控到地图中间重建（[26.1]）；
  - 是"自动检测闭合即结束"逻辑，建图中搬运若触发闭环检测则报"定位丢失"建图失败（[26.3 §2.4 Luba nRTK 款]）；
  - 不支持 8 字型地图；
  - AI 建图边界模糊就转手动，速度慢；
  - 视觉重定位动作非常磨草。
- 来源：[26.1]、[26.3]、[海外分享]

### 松灵 Yuka (含 nRTK 专项)

- **特色**：nRTK ↔ RTK 可手动切换、不需要重新建图；3 种定位方式可选；4G/Wi-Fi 信号源可选但不动态切换；nRTK 模式下断 4G & Wi-Fi 后仍可割草约 23min 才报"RTK 基站未就绪"。
- **短板**：首次进入界面默认非 nRTK 需手动切；nRTK 强绑定 4G（4G 过期则 nRTK 失效）。
- 来源：[26.1 §YUKA]、[海外分享]

### 松灵 Yuka Mini (视觉 + nRTK)

- **特色**：出界判定非常严格，1.2m 即报出界，需返回 0.4-0.6m 内才能启动。
- 来源：[26.3 §15 出界 - Yuka Mini]

### Sunseeker X7

- **短板**：地图边界不严格，建图离边界坑 ≥5cm 仍会掉，视觉无救；画禁区也掉。
- 来源：[26.1 §Sunseeker X7]

### Sunseeker X9 / Elite X9 Plus

- **特色**：硬件最豪华，15 传感器 + 8 摄像头（双目 + iTOF + 3 dTOF + 红外热成像）。
- **未解问题**：哪 8 个摄像头未明、dTOF FOV / 是否单点未明。
- 来源：[26.3 §竞品信息表]、[方案 §3.1 架构测试]

### Husqvarna Automower 560 EPOS

- **特色**：nRTK + 毫米波雷达 + Fleet management。
- **短板**：本批无定位实测。
- 来源：[26.3 §竞品信息表]

### 科沃斯 O1200

- **特色**：激光款 + 打草绳沿边；弓字调头速度快但掉头前谨慎。
- **短板**：斜坡边界无法建图、易跌落；打草绳人靠近误检（沿边一次中报 2 次 2m 内停下）。
- 来源：[26.3 §4.4 弓字 - 科沃斯]、[26.3 §4.5 沿边 - 科沃斯打草绳]
