## 2. 测试项清单（细粒度）

### L-T-001 激光建图回环精度（开阔场景）

- **场景 / 专题**：开阔场景激光建图
- **涉及竞品**：
  - 松灵 Luba（120° 面阵激光）：
    - 在开阔场景定位偏差明显；建图遥控走一圈回到原点偏差 ~2 个机身，**机器无法自动闭合**，需继续遥控顶围栏打滑校准才能保存（[海外分享]、[26.1 §松灵 - 建图]）
    - 厂商承认该问题，提供 RTK 选配 — 仅在特征点充足时面阵激光定位才好
- **关键指标**：开阔场景闭合误差 ≈ 2 个机身（**仅 Luba 单一定性数据**）
- **图片证据**：[26.1 §松灵 - 建图] img_v3_02u2_b4610f92...

### L-T-002 激光建通道时定位偏差

- **场景 / 专题**：通道建图（与 specials 通道专题关联）
- **涉及竞品**：
  - 松灵 Luba：建通道时遥控到基座面前但 App 显示机器没到、还差 1-2 个机身导致保存失败；连"设置通道起点"都失败；最终需遥控到地图中间重新建才能成功 [26.1 §松灵 - 建图]
- **关键指标**：通道场景下 App 显示位置与实际偏差 ≈ 1-2 个机身

### L-T-003 激光建图过程中搬运 / 拖拽（带 nRTK）

- **场景 / 专题**：建图鲁棒性
- **涉及竞品**：
  - 松灵 Luba：nRTK 信号好时搬运/拖拽都不会建图失败，轨迹和朝向**自动更新**；但 Luba 是"检测到闭合即自动完成"逻辑 → 搬运中若触发闭环检测则报"定位丢失"建图失败；**不支持 8 字型地图** [26.3 §2.4 Luba nRTK 款]、[方案 §4.1]
- **关键指标**：续建支持搬运 1m / 抬起拉拽；闭合误检导致建图失败

### L-T-004 激光重定位动作（关后台后）

- **场景 / 专题**：重定位
- **涉及竞品**：
  - 松灵 Luba：关闭后台后重新打开可续建 [26.1 §松灵 - 是否支持重定位续建]
  - 松灵 Luba：建图时搬动机器 → 建图失败（[26.1] 同节）
- **关键指标**：续建触发条件（关后台不致命）；搬动 + 建图态致命

### L-T-005 激光出界判定

- **场景 / 专题**：出界（已纳入）
- **涉及竞品**：
  - 九号 i2 激光款：慢推 3m 报错；抬后轮拉超 3m 放下后做重定位再正常回充 [26.3 §15 - 九号激光款 i2]
- **关键指标**：出界报错距离 ~3m；抬后轮拉拽场景 → 落地后主动重定位

### L-T-006 激光寻桩 / 充电桩位移（地图一致性）

- **场景 / 专题**：寻桩（已纳入）
- **涉及竞品**：
  - 松灵 Luba：只要红外能对接到就能回桩，但**地图充电桩位置不更新**；3 次实测：失败 → 成功 → 成功 [26.1 §松灵 - 充电桩移位后回充]
  - 松灵 Luba：移出充电桩后**没有自动回充动作**（九号 X3 / i 系列有）[26.1 §松灵 - 将机器移出充电桩]、[海外分享]
- **关键指标**：地图位置与实际位置脱节（已知短板）

### L-T-007 激光围困场景定位行为（已纳入）

- **场景 / 专题**：围困
- **涉及竞品**：
  - 松灵 Luba：被障碍物围困一圈，**未报定位异常**；预规划路线持续推进；尝试用"撞一下"二次确认是否真被困 [26.1 §松灵 - 围困视频]、[海外分享]
- **结论**：激光 + 视觉 + nRTK 组合下，围困不触发定位失效；靠脱困策略解

### L-T-008 激光纯定位精度（动态人前 + 脏污 / 遮挡）

- **场景 / 专题**：固态激光雷达定位扰动（**仅方案，未实测**）
- **涉及竞品**：方案文档明确"针对固态激光雷达产品"
  - 动态障碍（人）频繁经过相机/激光前方对定位精度的影响 [方案 §4.2]
  - 相机 / 激光雷达脏污或遮挡是否报错 [方案 §4.2]
- **状态**：**未实测**，列入 next-round。

### L-T-009 激光重定位 — 共视 / 无共视搬桩（**仅方案，未实测**）

- **场景 / 专题**：跨区域重定位
- **涉及竞品**：方案文档明确"仅激光雷达机型"
  - A 区域建图成功，搬桩到无共视的 B 区域割草 → 一段时间后触发重定位 → 是否会重定位到新 B 图上 [方案 §4.3.6]
- **状态**：**未实测**，列入 next-round。
