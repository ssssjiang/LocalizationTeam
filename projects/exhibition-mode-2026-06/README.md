## 相关技术决策

- 静态展位 = 2 台机器 + 2 台 GPU PC（防止观众遮挡）
- 双目算法跑在 PC 端而非板端（GPU + 15Hz vs 板端 7Hz）
- 雷达走独立网口、不挤 USB（双目 USB 167Mbps / 240Mbps 上限）
- 上位机选 RViz（Linux/Ubuntu）而非 QT（人力受限）
- **3D 地图素材渲染采用 Potree Web**（详见 howto）
