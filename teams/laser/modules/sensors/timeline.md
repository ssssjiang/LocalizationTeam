## 时间线

| 阶段 | 事件 | 结论 / 输出 | 来源 |
|------|------|-----------|------|
| 早期 | **需求提出**：割草机大场景 SLAM，目标 3000m²+；开始评估雷达型号 | 进入选型流程 | 割草机slam激光雷达需求 |
| 第一轮评估 | **速腾 Airy 内场联调**：发现点云拖影严重（玻璃鬼影、树叶噪点），SLAM 建图墙皮厚重、重影、Z 轴漂移 | 速腾 Airy 初步亮红灯 | 速腾airy和mid360 |
| 第一轮评估 | **向速腾厂商反馈**：获得回复——测距原理所致（光斑多帧叠加），暂时无法优化 | 速腾 Airy 淘汰（问题无法修复） | 速腾airy和mid360 |
| 第一轮评估 | **速腾安装角度调整尝试（前倾 -5°）**：引入地面点，高度漂移有所缓解，但建图质量仍差 | 调参无法根本解决问题 | 速腾airy和mid360 |
| 第一轮评估 | **Mid360 开源数据预评估**：人手持 1m 高度 rosbag，树轮廓清晰，无明显拖影，测距 70m+ | Mid360 表现符合预期 | 速腾airy和mid360 |
| 第一轮评估 | **Mid360 内部外场采集**：78号场地、105号场地数据采集，建图 + 定位验证通过，105场地颠簸数据稳定 | Mid360 外场验证通过 | 速腾airy和mid360 |
| 第一轮结论 | **初步选型 Mid360**：在速腾 Airy 确认淘汰后，Mid360 成为主要候选 | D-001 初步确立 | 速腾airy和mid360 |
| 第二轮评估 | **引入禾赛 JT16 做补充对比**：使用 105 号场地数据，与 Mid360 建图结果对比 | 新增候选评估 | 速腾airy和mid360+禾赛对比分析 |
| 第二轮评估 | **禾赛 JT16 树冠点云发散**：树冠处"发射状"点云，误差约 80cm，与速腾 Airy 类似的质量问题 | 禾赛 JT16 亮红灯 | 速腾airy和mid360+禾赛对比分析 |
| 第二轮结论 | **最终选定 Mid360**：宏观建图与禾赛接近，但微观点云质量更优（树干轮廓清晰、建筑表面均匀），三方对比中唯一通过 | 器件选型完成，D-001 最终确立 | 速腾airy和mid360+禾赛对比分析 |
| 第三轮评估 | **引入 Airy Lite / 万集16线 / 禾赛JT16 做新器件横向对比**：60/78/105场地基础建图、特殊场景（金字塔坡道、鹅卵石、窄通道、高反）、3000㎡验证、弓字定位稳定性、导轨精度 | 全量测试 | 010_新器件slam采集需求；015_新器件slam采集需求数据分析 |
| 第三轮评估 | **禾赛JT16 定量淘汰**：3000㎡多场景跑飞（10×10小圈、30×30弓字均 Failed）；弓字定位达标率73-80%，远低于产品要求；高反完全失效 | SD-003 第二轮证据补充，淘汰结论强化 | 015_新器件slam采集需求数据分析 |
| 第三轮评估 | **万集16线1200㎡可用，3000㎡有风险**：边界圈通过，10×10小圈 Failed；跨场地稳定性差（60/78场地达标率下探至80%级别）；高反 Failed | SD-005 | 015_新器件slam采集需求数据分析 |
| 第三轮结论 | **Airy Lite 推荐为新项目主候选**：3000㎡全场景通过，弓字达标率稳定在97-98%，高反/窄通道表现与 Mid360 相当；建图质量优于其他两款新器件 | SD-004 | 015_新器件slam采集需求数据分析 |
| 2025-08-04 | **Mid360 单体硬件测试完成**（Versa四驱机型）：量程/测距精度/FOV/高反/噪点/噪音/环境测试全部 Pass；白靶极限量程 ~80m，黑靶 ~40m（阳光下约 30m）；垂直FOV ~59.4°；噪音 28-36dB；高温加速寿命 402h Pass | SD-002 单体指标补充确认 | Mid360激光雷达单体测试报告_20260414 |
| 2025-11-24 | **Airy Lite 单体硬件测试开始**（速腾预研样品）：量程/FOV/环境测试/噪音基本通过；黑靶40m以远无点云（量程约30m）；强日照下白靶30m以远精度超标；雪天杂点 Fail；垂直FOV ~44.3°，更安静（21-22.5dB）；寿命测试(55°C) Ongoing（截至2026-04 已1700h+，持续通过） | SD-004 单体指标补充 | Airy Lite激光雷达单体测试报告_20260414 |
