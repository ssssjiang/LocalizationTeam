# 局部重定位减少对导航静止依赖优化



1. 短期防护：Set pose前后加个轮速，补偿到set pose结果上&#x20;

   1. 这个比较简单，现有框架下处理









2. 中期，没有主动重定位下： （现有框架下，无感的代码优化）

   1. Set pose开启线程独立计算；

   2. 同时，主线程用pose进行slam递推，正常计算和发布; 不刷新地图

   3. 局部重定位完成后

      1. 成功，修正初始pose；刷新地图

      2. 失败，返回false，导航会发起全局重定位；











* 长期，有主动重定位；  安全风险

  1. Set pose，**返回true，导航会开始走**；开启线程独立计算；

  2. 同时，主线程用pose进行slam递推，正常计算和发布；不刷新地图

  3. 局部重定位完成后

     1. 成功，修正初始pose；刷新地图

     2. 失败，**发起主动重定位**
