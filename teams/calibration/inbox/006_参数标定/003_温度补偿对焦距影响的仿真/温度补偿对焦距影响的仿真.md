# 温度补偿对焦距影响的仿真

# 1. 参考扫地机经验的计算

参考扫地机的经验，温度补偿的系数kx是

环境温度15度以下：0.0725

环境温度15度\~35度：0.07

环境温度45度以上：0.075

deltaT = currT - calibT

fx\_new = fx\_fac + kx \* deltaT

fy\_new = fy\_fac + kx \* deltaT



考虑两个极限工况

如果工厂温度是20度，工作温度70度（扫地机当50度处理），deltaT = 30度

如果工厂温度是30度，工作温度0度，deltaT = -30度

fx\_max = fx\_fac + 2.25

fy\_max = fy\_fac + 2.25



fx\_min = fx\_fac - 2.175

fy\_min = fy\_fac - 2.175
