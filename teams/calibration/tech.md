## 一、方案框架


| 维度         | 内容                                                                                                           | 来源                           |
| ---------- | ------------------------------------------------------------------------------------------------------------ | ---------------------------- |
| 解决什么问题     | 机器出厂/售后场景下，对前视双目相机、侧目相机（左右各一）、IMU 与 ODO 之间的内外参进行标定，保证定位系统输入参数准确                                              | 2025-10-23 售后标定设计文档；售后整机标定方案 |
| 主要输入       | 前视双目图像、侧目相机图像（640×544，鱼眼）、IMU 数据、ODO 数据、二维码特征点观测                                                             | 2025-10-23 售后标定设计文档；侧视相机参数定义 |
| 核心链路       | 机器在二维码场地走 S 曲线采集数据 → MSC Plugin 保存传感器数据 → Ceres BA 优化（重投影误差约束）→ 输出标定结果 → 写入 EEPROM + JSON 文件                 | 售后整机标定方案；定位组-售后双目基线标定开发计划    |
| 输出给谁       | EEPROM（双目内外参、双目基线 Tcrcl、侧目外参）；dualcameracalibration.json / dualcameraaftersalecalibration.json；供 SLAM 定位算法读取 | 2025-10-23 售后标定设计文档          |
| 对整体定位系统的价值 | 标定结果是 VSLAM 定位精度的基础，外参偏差直接影响建图与重定位质量                                                                         | —                            |
