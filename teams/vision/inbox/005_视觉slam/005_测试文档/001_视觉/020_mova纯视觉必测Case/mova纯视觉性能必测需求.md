# mova 纯视觉割草机 — 性能相关必测需求

> **目标读者**：现场测试人员  
> **测试对象**：mova 纯视觉机型（竞品，无法 USB 取机内数据）  
> **验证重点**：边界/覆盖精度、弱光与动态场景下的定位与行为、重定位与响应是否受算力制约（通过外部现象推断）

---

## 0. 通用约束与交付

### 0.1 约束

1. **禁止依赖 USB / 串口**：不读取机内日志与性能计数器。
2. **以现场观察与录像为准**：APP 轨迹仅作辅助；**出界/禁区以地面标线与地面实况为准**（与 Terramow Case 一致）。
3. **全程录像**：每个 Case 至少 1 段完整过程视频；定量项需保留慢动作/秒表画面。

### 0.2 建议携带工具


| 工具                           | 用途             |
| ---------------------------- | -------------- |
| 手机（支持 ≥120fps，优先 240fps 慢动作） | P1 响应延迟        |
| 秒表 / 手机计时                    | P4、建图回桩耗时      |
| 卷尺 / 皮尺                      | 漏割宽度、出界距离      |
| 照度计                          | 低照度场景记录        |
| 红外测温枪（可选）                    | P5 机身温度        |
| 石灰粉 / 喷漆 / 低拉线（不高于草坪）        | 边界、禁区、10m 方框标线 |


### 0.3 交付物（每个 Case）

- 测试日期、天气、大致照度（如有照度计则写数值）
- 步骤是否按文档执行（是/否，若否说明偏差）
- **结论**：通过 / 不通过 / 待复测 + 一句话原因
- 照片或视频文件名（与网盘/飞书目录一致）

## 1. 出界检查（定性）

**对应参考**：Terramow `1.2.1 出界检查`

### 1.1 场地要求

1. 建图时边界**不要**完全贴死栅栏、建筑等硬边界，留出可观察余量。
2. 地图边缘用**地面标线**（画线或拉线，**不高于草坪**）标出「名义边界」。
3. 建图成功后保存地图。

### 1.2 执行步骤

1. 下发正常割草任务，覆盖完整工作区域至少 **1 轮**（若产品支持多轮则做 1 轮即可）。
2. 全程跟机观察：刀盘/机身是否越过地面标线。

### 1.3 记录项


| 记录项    | 说明                              |
| ------ | ------------------------------- |
| 是否出界   | 是 / 否                           |
| 最大出界位置 | 用文字描述相对标线的方位                    |
| 最大出界距离 | 用卷尺量「越线最远处」到标线的垂直距离（cm），若未出界填 0 |
| 证据     | 出界点**地面+机身**同框照片；或视频时间点         |


---

## 2. 禁区检查（定性）

**对应参考**：Terramow `1.2.2 禁区检查`

### 2.1 场地要求

1. 在地图**中间**用地面标线围出一块禁区（矩形即可，边长建议 ≥1m）。
2. 在 APP 中配置对应禁区（若产品支持；若不支持禁区则本项记「不适用」并说明）。

### 2.2 执行步骤

1. 下发割草任务，覆盖包含禁区的区域至少 **1 轮**。
2. 观察机身/刀盘是否进入禁区标线内。

### 2.3 记录项


| 记录项       | 说明               |
| --------- | ---------------- |
| 是否冲入禁区    | 是 / 否            |
| 最大侵入位置与距离 | 同出界，单位 cm；未侵入填 0 |
| 证据        | 照片 / 视频时间点       |


---

## 3. 调头漏割（定量）

**对应参考**：Terramow `2.2.1 调头漏割`（10m×10m 外框，弓字长度 10m）

### 3.1 场地要求

1. 遥控或沿边建立 **10m×10m** 外框地图，弓字路径长度约 **10m**（与 Terramow 示意图一致）。
2. 地面标线标出外框，便于测量漏割。

### 3.2 执行步骤

1. 在该地图内执行一次完整覆盖作业。
2. 作业结束后，检查**调头附近**未割草条带。

### 3.3 记录项


| 记录项      | 说明              |
| -------- | --------------- |
| 调头漏割最大宽度 | 多条则取最大，单位 cm    |
| 典型漏割条带数量 | 整数              |
| 证据       | 俯拍或蹲拍照片，必要时卷尺入镜 |


---

## 4. 建图流程与回桩耗时（行为 + 性能）

**对应参考**：Terramow `二、建图测试` 第 1 条

### 4.1 执行步骤

1. **清空或新建地图**（按产品能力），完整走一遍「从零到建图完成」流程，**录像**。
2. 观察并文字记录：
  - 建图路径形态（稀疏弓字 / 密弓字 / 沿边圈数等）。
  - 建图过程中遇障碍时的行为（频繁停走、绕障、报错等）。
3. 建图结束回桩后：
  - **回桩前**截 APP 地图一张；
  - 回桩后等待直至地图无明显变化（或等待 **60s** 取先到者），再截一张；
  - 用秒表记录：**回桩完成到可开始下一次操作**之间，若出现「地图保存」「数据处理」等提示或转圈，分别记各阶段**可见耗时**（秒）。

### 4.2 记录项


| 记录项       | 说明                                |
| --------- | --------------------------------- |
| 建图总耗时     | 从开始建图到建图成功提示，单位 min               |
| 回桩后地图是否变化 | 是 / 否，若变化附两张截图文件名对比               |
| 回桩后处理阶段耗时 | 列出各阶段名称与秒数；若无阶段提示则写「无拆分，总等待 xx s」 |
| 异常现象      | 卡死、反复失败、丢图等，若有则描述并附视频             |


---

## 5. 低照度工作能力（定量 + 行为）

**对应参考**：Terramow `三、定位精度测试` 第 1 条（照度计）

### 5.1 执行步骤

1. 在**可作业时段内**，从照度较高开始，选择 **≥3 个**照度档位（建议包含：阴天傍晚、日落后不久、产品允许作业的最低环境）。
2. 每个档位：
  - 用照度计在**机身高度、大致朝向相机**的位置测照度，记录 **lux**；
  - 尝试**开始建图或开始割草**（以产品当时允许的操作为准）。
3. 若产品存在「日出日落强制限制」，如实记录**软件提示文案**与**系统时间**，不强行破解。

### 5.2 记录项


| 记录项           | 说明                                 |
| ------------- | ---------------------------------- |
| 各档位 lux 与能否启动 | 表格：lux / 能否启动 / 提示语                |
| 最低可工作照度       | 能稳定开始作业的最小 lux（若测不到则写「受软件时段限制未测到」） |
| 证据            | 照度计读数入镜照片或视频                       |


---

## 6. 动态障碍物下定位与行为

**对应参考**：Terramow `三、定位精度测试` 第 4 条

### 6.1 执行步骤

1. 在**割草作业进行中**，测试人员以正常步行速度在机器**相机前方**反复横穿（注意安全，保持侧向距离可控）。
2. 持续干扰 **≥3 min** 或直至机器报警/停机（先到者）。
3. 干扰结束后，继续观察机器 **≥5 min** 是否正常作业、是否偏航、是否停走异常。

### 6.2 记录项


| 记录项    | 说明                             |
| ------ | ------------------------------ |
| 避让是否正常 | 是 / 否 / 部分失效                   |
| 定位是否异常 | 如：轨迹乱跳、原地转圈、漏割加剧、APP 位置与实际明显不符 |
| 报警或停机  | 若有，记录**完整中文报错文案**              |
| 证据     | 全程视频文件名                        |


---

## 7. 抱起后重定位（行为 + 耗时）

**对应参考**：Terramow `四、重定位测试` 第 1 条 + 性能项 **P4**

### 7.1 执行步骤

1. 在已建图且可正常作业状态下，**抱起机器**（按产品允许方式），在**同一块草坪内**重新放置于一新位置（与原地水平位移建议 **1～3m**）。
2. 秒表从「放下机器」开始计时，到「机器恢复按地图正常割草（无持续重定位转圈）」结束，记为 **重定位恢复耗时**。
3. 重复 **≥3 次**（每次放置位置可略有不同），取记录表。

### 7.2 记录项


| 记录项      | 说明                          |
| -------- | --------------------------- |
| 重定位动作描述  | 如：直行、转圈、组合；是否反复失败           |
| 每次恢复耗时   | 单位 s，至少 3 次                 |
| 恢复后位置一致性 | APP 与实地大致是否一致（主观：一致 / 偏差明显） |
| 证据       | 至少 1 次全程含秒表画面视频             |


---

## 8. 障碍物响应延迟（定量，性能 P1）

**对应参考**：本轮新增性能项 **P1**

### 8.1 执行步骤

1. 机器以**典型作业速度**直线前行（通道或开阔地均可，保证安全余量）。
2. 当机器前端距障碍物约 **1m** 时，由侧方人员将**低矮软障碍**（如纸箱）快速移入路径（**勿挡人**，侧方操作）。
3. 使用 **≥120fps，优先 240fps** 录制障碍进入画面与机器反应。
4. 重复 **≥5 次**（每次间隔待机器恢复常态路径）。

### 8.2 记录与换算

1. 在剪辑软件或逐帧播放中，数帧：
  - **T0**：障碍完整进入刀盘前方安全可视范围的首帧；
  - **T1**：机器明显减速或刀盘停转或开始转向的首帧。
2. 延迟 **ms = (T1 − T0) / 帧率 × 1000**。
3. 填写每次结果与 **5 次平均值**。

### 8.3 记录项


| 记录项  | 说明       |
| ---- | -------- |
| 录制帧率 | 如 240fps |
| 每次延迟 | 单位 ms    |
| 平均延迟 | 单位 ms    |
| 证据   | 原始慢动作文件名 |


---

## 9. 长时作业与热表现（性能 P5）

**对应参考**：本轮新增性能项 **P5**

### 9.1 执行步骤

1. 满电或高电量开始，执行**连续作业 ≥3 h**（若电量不足则记实际连续时长，并说明）。
2. 在 **T0（开始）**、**T1（1 h）**、**T2（3 h 或结束）** 三个时刻：
  - 可选：用红外测温枪测**机身主控附近外壳**温度（若无量化则手写「温/烫/可手触」）。
  - **重复 Case 8（P1）1 次**（同一障碍方案、同帧率），记录延迟 ms。
3. 观察全程是否出现：**明显卡顿、频繁停走、APP 严重滞后、异常报警**。

### 9.2 记录项


| 记录项                  | 说明          |
| -------------------- | ----------- |
| 连续作业时长               | 单位 h        |
| 各时刻外壳温度或主观热感         | 表格          |
| T0 / T1 / T2 的 P1 延迟 | 各 1 次 ms    |
| 长时异常摘要               | 无则写「无」      |
| 证据                   | 长录像分段文件名或说明 |
