# 新员工开发文档

# 一、入职必过

PS：因开发节奏很快，以下信息随时可能更新，新入职同学有问题请及时联系入职伙伴，并且持续更新新人文档

1. 环境搭建[ 办公 - 新人warmming-up](https://roborock.feishu.cn/docx/H6KkdOmMGo5N1oxyO5KcZjCJnJe)

   1. 外网开发机申请

      1. 配置

      > 处理器：U9 285K；显卡：RTX4070（若无法采购，可更换为RTX4080S）；内存：32G\*2；存储：机械硬盘2T+固态硬盘2T；配件：显示器+键鼠一套

      * 系统：ubuntu 22.04

   2. 注意开通gitlab2\3\1\5，并将用户名、gitlab链接改为姓名的拼写

   3. 注意内外网机器，特别是内网机环境配置好后再联系IT锁机箱

2. 权限申请[ 割草机SLAM代码库及构建输出目录](https://roborock.feishu.cn/wiki/DKZbwZEECiUefykVn4Qcwgxknkc)

   1. gitlab相关代码库按照方向申请

      1. 先申请外网gitlab代码权限

      2. 联系IT热线给内网机配网、上锁

      3. 申请内网gitlab代码权限

   2. share按照按照方向申请

      1. share目前要申请的都是外网的，可与gitlab同步申请

   3. 自动化工具后续逐步申请

3. 代码编译与仿真

   1. 走通[ 割草机仿真和上机流程](https://roborock.feishu.cn/wiki/J3wyw6uiviGUeUkD4X6ceCXAnCd)的仿真一节

   2. vslam方向同学，走通[ 编译vslam](https://roborock.feishu.cn/wiki/QGYCwrSA4iZdslkKBsTcQAi6nxc)

4. 代码架构理解

   1. [ 架构文档](https://roborock.feishu.cn/wiki/G318wn5AZi2qqFkWtUrc0e3Hn9c)

5. 代码提交学习

   1. [割草机Plugins/Controller代码介绍](https://roborock.feishu.cn/minutes/obcn5z94sz27ebh7e28e918q)

   2. [割草机仿真和上机流程](https://roborock.feishu.cn/minutes/obcn4q5l6ppu2xoxatl779ro?q=%25E6%259E%2597%25E5%25AD%2590%25E8%25B6%258A)

   3. [代码提交流程介绍](https://roborock.feishu.cn/minutes/obcn5awnhaonk844lp5rp164)

      1. 提PR时，分支名都以如下的格式提代码：private/xxxxxxx/bbbbbb

      2. 其中xxxxx为姓名/姓名缩写，bbbb为表示此次修改的标注

   4. [禅道bug解决流程](https://roborock.feishu.cn/minutes/obcni56746191nba7nn8j645)
