# 一、水下机器人常用传感器


| 传感器                               | 测量值      | 优点                | 缺点                           |

| 多普勒速度计(Doppler Velocity Log, DVL) | 相对于水流的速度 | 提供速度量，可以用于递推      | 价格昂贵；易受水流速度影响                |
| 激光/tof/结构光                        |          |                   | 水下可能量程较短；室外可能受阳光影响           |

| GNSS                              |          | 水面可以使用；水下可以使用水面浮标 | 水下信号弱；受水流延迟，计算值可能不准          |
| 水深计                               |          |                   |                              |

# 二、泳池传感器方案

1. 纯水面清洁：

可以复用RTK+imu方案

- 纯水底清洁：

可以复用2D激光/3D tof方案，需要加一个递推传感器 

- 水面+水底+侧壁
  1. 地图：3D地图+底层2D地图
  2. 水面沿边：
    1. tof+imu，缺少一个可靠递推传感器
    2. 向下超声可能只能建图，不能用来定位
  3. 水底定位：
    1. 复用2D激光/3D tof方案，建2D/3D地图，在平面上导航
  4. 侧壁定位：
    1. 纯递推+水深计
    2. 机器向上时前向tof会打出水面，向左/右或向下时，只有单边有观测，很难做激光定位
      Image Token: GtDAbrrYtoov2axPS4ocEvBtniz](images/GtDAbrrYtoov2axPS4ocEvBtniz.png)
  5. 重定位：
    1. 机器每次可能从水面上随机位置启动
    2. 规则形状泳池可能无法重定位
    3. 从侧壁回到水底，可能需要做重定位
